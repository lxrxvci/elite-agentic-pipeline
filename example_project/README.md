# Elite Freelancer SaaS Dashboard

A reference implementation generated by the **Elite Agentic SDLC Pipeline**.

## What it is

A full-stack freelancer dashboard for tracking time, creating invoices, and getting paid faster. It demonstrates the Golden Path stack and elite delivery practices:

- **Frontend:** Next.js 15, React 19, TypeScript, Tailwind CSS
- **Backend:** FastAPI, SQLAlchemy 2.0, Alembic, Pydantic v2
- **Database:** PostgreSQL (Neon/RDS in production, Docker locally)
- **Auth:** Clerk OIDC/OAuth2 in production, dev-only JWT locally
- **Hosting:** Vercel (frontend + backend serverless functions)
- **CI/CD:** GitHub Actions with security scanning, contract tests, and Vercel deployments
- **Observability:** OpenTelemetry + Grafana Cloud in production; Prometheus/Grafana/Tempo/Loki locally

## Quick start

See [`GETTING_STARTED.md`](./GETTING_STARTED.md).

```bash
cp .env.example .env
make setup
make ci
```
