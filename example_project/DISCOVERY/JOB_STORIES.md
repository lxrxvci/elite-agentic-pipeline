# Job Stories

*Jobs-to-be-Done stories for freelancers using the dashboard. Format: When [situation], I want to [motivation], so I can [expected outcome].*

---

1. **When I finish a block of client work,** I want to log that time in one click from where I already am, so I can bill accurately without breaking my flow.

2. **When it is time to invoice at the end of the week,** I want all my unbilled hours grouped by client and project, so I can create an invoice in under a minute.

3. **When I send an invoice,** I want the client to be able to pay instantly with their preferred method, so I can shorten the time between sending and getting paid.

4. **When an invoice is approaching its due date,** I want a polite reminder to go out automatically, so I can avoid an awkward personal follow-up and still get paid on time.

5. **When I log in on Monday morning,** I want to see exactly who owes me money and for how long, so I can prioritize my follow-ups and reduce cash-flow anxiety.

6. **When a client asks for a “quick” out-of-scope task,** I want to capture it immediately before I forget, so I can bill for the extra work instead of eating the cost.

7. **When I review my month,** I want to see which clients and projects generated the most revenue, so I can decide where to invest my time and when to raise rates.
